# Privacy Policy

This extension uses the browser's Local Storage API only to save user settings and local state within the user's browser.

No personal information or user data is collected, transmitted, shared, or stored on external servers.

All data remains locally on the user's device and is never sent outside the browser.

The extension does not use analytics, tracking, advertising, or third-party data collection services.

---

# プライバシーポリシー

本拡張機能は、ユーザー設定およびローカル状態の保存のために、ブラウザの Local Storage API のみを使用します。

個人情報やユーザーデータを収集、送信、共有、または外部サーバーへ保存することはありません。

すべてのデータはユーザーのブラウザ内および端末上にのみ保存され、外部へ送信されることはありません。

また、解析・トラッキング・広告・第三者によるデータ収集サービス等は利用していません。
